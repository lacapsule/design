FTP

File transfer protocol

Permet de transférer des fichiers de toutes natures

Trivial file transfer protocol

SFTP

FTPS

Permet de transférer des fichiers de manière sécurisée

SCP

Secure copy protocol

Permet de transférer des fichiers sous une couche de chiffrement
